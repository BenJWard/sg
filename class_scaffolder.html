The Scaffolder object provides functions to locate or resolve regions of repetitions and haplotype originated bubbles. It requires a valid SequenceGraph, a vector of ReadMappers and a KmerCompressionIndex.

Parameters
_sgA SequenceGraph to scaffold
_rmsA vector of PairedReadmapper, containing the mapping of reads to _sg
_lrmsA vector of LinkedReadMapper, containing the mapping of linked reads to _sg
_kciA KMerCompressionIndex object
